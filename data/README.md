# Data Folder — Traffic Sign Classification Project

This folder is for storing all raw datasets used in the project.  

## Getting the Data

We use three Kaggle datasets:

1. [Traffic Sign Dataset Classification](https://www.kaggle.com/datasets/ahemateja19bec1025/traffic-sign-dataset-classification)  
2. [Traffic Signs Preprocessed](https://www.kaggle.com/datasets/valentynsichkar/traffic-signs-preprocessed)  
3. [Traffic Signs Dataset](https://www.kaggle.com/datasets/tuanai/traffic-signs-dataset)  

> **Important:** Do NOT store your `kaggle.json` API key in this repo.
